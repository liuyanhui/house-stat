# 北京房地产数据抓取程序

从北京市住建委网站自动抓取房地产签约数据，包括新房和二手房的月度统计、每日签约数据、五年历史数据等。

[![Python](https://img.shields.io/badge/Python-3.x-blue.svg)](https://www.python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 功能特性

### 二手房（存量房）数据
- ✅ 经纪机构月度排行（发布套数、签约套数、退房套数）
- ✅ 区县月度统计（签约套数、成交面积）
- ✅ 面积区间月度统计
- ✅ 价格段月度统计
- ✅ 每日网上签约数据
- ✅ 月度汇总数据
- ✅ 近五年交易情况

### 新房（商品房）数据
- ✅ 每日数据统计（可售期房、未签约现房、期房认购/签约等）
- ✅ 近五年网签情况

### 通用功能
- ✅ 智能去重：避免重复抓取相同数据
- ✅ 完整日志：记录所有操作和数据状态
- ✅ 数据转置：自动将横向表格转换为竖向格式
- ✅ 异常处理：网络失败自动重试
- ✅ 向后兼容：自动扩展CSV列，保留历史数据

## 数据来源

- **数据源**：北京市住房和城乡建设委员会
- **URL**：http://bjjs.zjw.beijing.gov.cn/eportal/ui?pageId=307749
- **数据类型**：存量房（二手房）、新建商品房（新房）
- **更新频率**：每日更新

## 使用方法

### 1. 基本使用

直接运行程序：

```bash
python house_stat.py
```

程序会自动：
1. 抓取最新网页数据
2. 提取数据年月和日期
3. 解析各类数据
4. 保存到CSV文件（如已存在则跳过）
5. 输出统计信息

### 2. 首次运行

首次运行会创建以下文件：

**二手房数据：**
- `agency_monthly.csv` - 经纪机构月度数据
- `district_monthly.csv` - 区县月度数据
- `area_monthly.csv` - 面积区间月度数据
- `price_monthly.csv` - 价格段月度数据
- `resale_daily.csv` - 每日网上签约数据
- `resale_monthly.csv` - 月度汇总数据
- `resale_5year.csv` - 近五年交易情况

**新房数据：**
- `new_daily.csv` - 每日商品房数据统计
- `new_5year.csv` - 近五年网签情况

**其他：**
- `house_stat.log` - 运行日志

### 3. 后续运行

程序会自动检查数据是否已存在：
- 如果数据已存在：记录日志并跳过
- 如果数据不存在：追加新数据

建议每日或定期运行程序以获取最新数据。

```python
MAX_RETRIES = 3
RETRY_DELAY = 5  # 秒
```

## 注意事项

1. **数据时效性**：
   - 月度数据：页面显示的是上一个月的完整数据
   - 每日数据：页面显示的是最新日期的签约数据
   - 五年数据：每年更新，覆盖历史数据

2. **网络要求**：需要能够访问北京市住建委网站

3. **文件编码**：CSV文件使用UTF-8 with BOM编码，Excel可直接打开

4. **数据去重**：程序会自动检查并跳过已存在的数据

5. **默认值处理**：数据抓取失败时，该字段值设为 -1

6. **向后兼容**：CSV文件列数增加时，自动备份并扩展现有数据

## 常见问题

### Q1: 程序报错"未找到表格"怎么办？
A: 可能是网站结构发生变化，请查看实际网页结构并修改代码中的表格ID或查找逻辑。

### Q2: 如何查看历史月份数据？
A: 程序每次运行都会追加新数据到CSV文件，可以直接在CSV中查看所有历史记录。

### Q3: 每日数据会重复抓取吗？
A: 不会。程序会检查日期，如果同日期的数据已存在，会自动跳过。

### Q4: 可以设置定时自动运行吗？
A: 可以使用Windows任务计划程序或Linux的cron来设置定时任务。

### Q5: 为什么有些字段显示 -1？
A: 当数据抓取失败或网页中不存在该字段时，默认值设为 -1，便于后续数据处理。
