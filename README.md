# 北京房地产数据抓取程序

从北京市住建委网站自动抓取房地产签约数据。

[![Python](https://img.shields.io/badge/Python-3.x-blue.svg)](https://www.python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 功能

**二手房数据**：经纪机构排行、区县/面积/价格段统计、每日/月度数据、五年历史

**新房数据**：每日签约统计、五年网签数据

**特性**：智能去重、自动重试、日志记录、向后兼容

## 数据来源

北京市住建委：http://bjjs.zjw.beijing.gov.cn/eportal/ui?pageId=307749

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
python main.py
```

程序自动抓取数据并保存到 `data/` 目录，已存在的数据会自动跳过。

数据分析：

```bash
python script/analyze.py
```

### `data/` — 数据输出目录

所有抓取的房地产数据以 CSV 格式存储于此。CSV 使用 UTF-8 with BOM 编码，Excel 可直接打开。

#### 二手房（存量房）数据

| 文件 | 说明 | 列 |
|------|------|----|
| `resale_daily.csv` | 二手房每日网上签约数据 | 日期, 签约套数, 签约面积, 住宅签约套数, 住宅签约面积, 可售房源套数, 可售房源面积, 可售住宅套数, 可售住宅面积, 新发布房源套数, 新发布房源面积, 新发布住宅套数, 新发布住宅面积 |
| `resale_monthly.csv` | 二手房月度网上签约汇总 | 月份, 网上签约套数, 网上签约面积(m2), 住宅签约套数, 住宅签约面积(m2) |
| `agency_monthly.csv` | 经纪机构月度签约排行 | 年月, 序号, 经纪机构, 发布套数, 签约套数, 退房套数 |
| `district_monthly.csv` | 按区县统计的月度签约数据 | 年月, 区县, 签约套数, 成交面积 |
| `area_monthly.csv` | 按面积区间统计的月度签约数据 | 年月, 面积区间, 成交套数, 成交面积 |
| `price_monthly.csv` | 按价格区间统计的月度签约数据 | 年月, 价格区间, 发布套数, 发布面积, 成交套数, 成交面积 |
| `resale_5year.csv` | 近五年存量房交易年度数据 | 年份, 住宅套数万, 住宅面积万m2, 非住宅面积万m2, 数据更新日期 |

#### 新房（商品房）数据

### `log/` — 日志目录

| 文件 | 说明 |
|------|------|
| `house_stat.log` | 程序运行日志，记录抓取过程、解析结果、错误信息 |

### `parsers/` — 数据解析模块

| 文件 | 说明 |
|------|------|
| `__init__.py` | 模块导出，统一暴露所有解析函数 |
| `base_parser.py` | 基础辅助函数：类型转换（`safe_int`, `safe_float`）、数据年月提取、日期计算 |
| `monthly_parsers.py` | 月度数据解析：经纪机构排行、区县分布、面积区间、价格区间、月度汇总、五年历史 |
| `daily_parsers.py` | 每日数据解析：二手房每日签约、商品房每日统计 |

### `utils/` — 工具模块

| 文件 | 说明 |
|------|------|
| `__init__.py` | 模块导出，统一暴露所有工具函数 |
| `fetcher.py` | 网页抓取：HTTP 请求、失败自动重试（最多 3 次） |
| `storage.py` | 数据存储：CSV 写入（自动去重）、控制台展示新增数据、列扩展向后兼容 |
| `directory.py` | 目录管理：确保 data/ 和 log/ 目录存在 |
| `logging_setup.py` | 日志配置：同时输出到文件和控制台 |

### `script/` — 分析脚本

| 文件 | 说明 |
|------|------|
| `analyze.py` | 北京二手房成交量综合分析：月度概览、周度分析、工作日/休息日对比、区县/面积分布、同比环比趋势 |

## 注意

- CSV使用UTF-8 with BOM编码，Excel可直接打开
- 数据抓取失败时，字段值为 -1
- 可设置定时任务自动运行（Windows任务计划或Linux cron）

## 常见问题

**Q: 程序报错"未找到表格"？**
A: 网站结构可能变化，需检查网页并修改代码

**Q: 如何查看历史数据？**
A: 直接打开CSV文件查看所有记录

**Q: 会重复抓取吗？**
A: 不会，程序自动跳过已存在数据
